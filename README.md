# SnakeGame

Jest to prosta gra w **Snake** napisana w języku **C#**. Gra jest uruchamiana w konsoli, a celem gracza jest kontrolowanie węża, który zjada jedzenie, rośnie i unika przeszkód oraz ścian.

## Jak uruchomić

1. Sklonuj repozytorium:
    ```bash
    git clone https://github.com/MishaNefedkin/SnakeGame.git
    ```

2. Otwórz projekt w **Visual Studio** lub **JetBrains Rider**.

3. Zbuduj projekt i uruchom go.

## Sterowanie

- Strzałki do poruszania wężem:
  - **↑** - w górę
  - **↓** - w dół
  - **←** - w lewo
  - **→** - w prawo

## Cele gry

- Zjedz jak najwięcej jedzenia, aby zwiększyć długość węża.
- Unikaj kolizji ze ścianami oraz z własnym ciałem.
- Gra kończy się, gdy wąż zderzy się z przeszkodą lub ze sobą.

## Współpraca

Projekt jest przeznaczony do wspólnej pracy, w której uczestnicy dzielą się zadaniami, np. implementacją ruchu węża, przeszkód, punktacji itd. Każdy uczestnik może pracować w swojej gałęzi i wysyłać zgłoszenia zmian do głównej gałęzi repozytorium.
